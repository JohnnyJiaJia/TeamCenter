社团中心 - TeamCenter
==========
一个社团管理系统，面向技术开发类社团，基于ThinkPHP开发。
#前言
##起源
最初璨看到社团内用Q群来通知信息，觉得这种方法非常笨重，而且不易于组员交流、讨论，在于刘伟做功能上的商议以后，于是萌发了动手写这么一个平台的想法，前前后后做了十几天，终于大致做完。
写的非常潦草，也请见谅。
